# Image Files for Equine Ownership Portfolio

Place all images of Billy in this directory with the following naming convention:

## Required Images

1. **billy-front-1.jpg** - Front view of Billy showing chestnut coat, white blaze, and white socks
   - Used in: Home (hero), About Billy (main photo), About Billy (gallery)

2. **billy-front-2.jpg** - Additional front view of Billy
   - Used in: About Billy (additional photo)

3. **billy-side-1.jpg** - Side profile view of Billy showing conformation
   - Used in: About Billy (side profile), Conformation (lateral view), Health & Wellness (BCS photo)

4. **billy-side-2.jpg** - Additional side profile view
   - Used in: About Billy (gallery), Conformation (additional lateral view)

5. **billy-rear.jpg** - Rear view showing hindquarters, white socks, and leg conformation
   - Used in: Conformation (rear view)

6. **billy-hindquarters.jpg** - Close-up view of hindquarters and back showing topline
   - Used in: Conformation (additional view)

## Image Specifications

- Format: JPG or PNG
- Recommended resolution: At least 1200px on the longest side
- File size: Optimize for web (under 2MB per image recommended)

## Usage

Once images are placed in this directory, they will automatically appear throughout the website. The images are referenced using paths like `/images/billy-front-1.jpg`.
