# Image Placement Guide

This guide shows where each photo of Billy is used throughout the website.

## Image Files Needed

Place these 6 images in the `/public/images/` directory:

### 1. billy-front-1.jpg
**Description:** Front view of Billy showing chestnut coat, white blaze, and white socks  
**Used in:**
- **Home Page:** Hero section background image
- **About Billy Page:** Main photo (right column)
- **About Billy Page:** Image Gallery (first image)

### 2. billy-front-2.jpg
**Description:** Additional front view of Billy  
**Used in:**
- **About Billy Page:** Additional photo (right column, third image)

### 3. billy-side-1.jpg
**Description:** Side profile view showing conformation  
**Used in:**
- **About Billy Page:** Side profile view (right column, second image)
- **About Billy Page:** Image Gallery (second image)
- **Conformation Page:** Lateral view (middle of three views)
- **Health & Wellness Page:** BCS assessment photo

### 4. billy-side-2.jpg
**Description:** Additional side profile view  
**Used in:**
- **About Billy Page:** Image Gallery (third image)
- **Conformation Page:** Additional lateral view (bottom section)

### 5. billy-rear.jpg
**Description:** Rear view showing hindquarters, white socks, and leg conformation  
**Used in:**
- **Conformation Page:** Rear view (right of three main views)

### 6. billy-hindquarters.jpg
**Description:** Close-up view of hindquarters and back showing topline  
**Used in:**
- **Conformation Page:** Additional view (bottom section, left image)

## Quick Setup

1. Copy all 6 image files to `/public/images/` directory
2. Name them exactly as listed above
3. The website will automatically display them in the correct locations

## Image Requirements

- **Format:** JPG or PNG
- **Resolution:** Recommended 1200px+ on longest side
- **File Size:** Optimize for web (under 2MB each recommended)

## Notes

- All images are clickable and will open in a lightbox for full-size viewing
- Images include proper alt text for accessibility
- Images are responsive and will scale appropriately on all devices
